# Установка из исходного кода

Выполнить в консоли:

```shell
$ cd ./seeder
$ go install ./cmd/...
```

Проверить, вызвав:

```shell
$ seeder -h
```

!!! example

    ```
    Usage of seeder:
      -c string
            config file (default "./seeder.yml")
      -seeder-help string
            show seeder help (ex: -seeder-help kafka)
    ```
